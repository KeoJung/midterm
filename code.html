HTML 코드 설명

아래는 HTML 코드의 주요 구조와 각 요소의 역할에 대한 설명입니다.

헤더

            
              <header>
                <h1>코드 설명 페이지</h1>
              </header>

헤더는 페이지의 상단에 위치하며, 페이지의 제목을 표시합니다. `< h1 >` 태그는 가장 중요한 제목을 나타내며, 검색 엔진 최적화(SEO)에도 중요한 역할을 합니다.

네비게이션 바

            
              <nav>
                <a href="./jungyeonseo.html">Introduce</a>
                <a href="./study.html">Study</a>
                <a href="./secon.html">SECON</a>
                <a href="./code.html">Code 설명</a>
              </nav>

네비게이션 바는 페이지 간 이동을 위한 링크를 제공합니다. `< a >` 태그는 하이퍼링크를 생성하며, `href` 속성을 통해 연결할 페이지의 경로를 지정합니다.

푸터

            
              <footer>
                <p>© 2025 웹프로그래밍 2조 | 코드 설명 페이지</p>
              </footer>

푸터는 페이지의 하단에 위치하며, 저작권 정보나 추가적인 링크를 표시하는 데 사용됩니다. `< p >` 태그는 텍스트를 단락으로 표시합니다.

CSS 코드 설명

아래는 CSS 코드의 주요 스타일과 각 스타일의 역할에 대한 설명입니다.

배경 스타일

            
              body {
                background-image: url('background.jpg');
                background-size: cover;
                background-repeat: no-repeat;
              }

이 스타일은 페이지의 배경에 이미지를 추가합니다. `background-image`는 배경 이미지를 설정하며, `background-size: cover`는 이미지를 화면 크기에 맞게 조정합니다. `background-repeat: no-repeat`은 이미지가 반복되지 않도록 설정합니다.

네비게이션 링크 스타일

            
              nav a {
                text-decoration: none;
                color: black;
                margin-right: 10px;
              }

이 스타일은 네비게이션 바의 링크를 꾸밉니다. `text-decoration: none`은 링크의 밑줄을 제거하고, `color: black`은 글자 색상을 검정으로 설정합니다. `margin-right: 10px`은 링크 간 간격을 추가합니다.
